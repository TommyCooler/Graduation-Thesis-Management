# Subject Service - Gateway Endpoints

Gateway base URL: `http://localhost:8080`

Service route prefix: `/subject-service`

## Subjects
- GET    `/subject-service/api/subjects/{subjectID}`
- PUT    `/subject-service/api/subjects/{subjectID}`
- DELETE `/subject-service/api/subjects/{subjectID}`

## Departments
- GET    `/subject-service/api/departments/{departmentID}`
- PUT    `/subject-service/api/departments/{departmentID}`
- DELETE `/subject-service/api/departments/{departmentID}`
